# guess_game
## Rules
- A secret number is chosen.
- The number is within a fixed range.
- The player guesses the number.
- After each guess, a hint is given.
- The game ends when the correct number is guessed.

## Requirements
- Secret number
- Number range
- Player input
- Hint system
- Guess checking logic

## How to Run
- Start the game.
- Set the number range.
- Choose the secret number.
- Ask the player to guess.
- Check the guess.
- Display hint.
- Repeat until correct.
- End the game.
